# web-admin
Administration for new GUG.cz web

## Instalace ##

naklonujte si repozitář a v root složce projektu spusťte

```
npm install
```

pro vývoj pak

```
npm start
```


### Build pro nasazení ###
```
npm run buil
```
zbuilduje aplikaci do několika statických souborů ve složce /public
